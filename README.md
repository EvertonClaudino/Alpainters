## 📋 Sobre o Projeto

Este repositório contém o código-fonte do website institucional da **Alpainters**, empresa portuguesa com mais de 10 anos de experiência em alpinismo industrial. O site apresenta os serviços, projetos realizados e meios de contacto da empresa, com foco em performance, acessibilidade e design moderno.

---

## ✨ Funcionalidades

- **Página única (SPA)** com navegação suave entre secções
- **Carrossel de imagens** com apresentação dos principais serviços
- **Galeria de projetos** com portfólio de trabalhos recentes
- **Formulário de contacto** integrado para pedidos de orçamento
- **Design responsivo** adaptado para mobile, tablet e desktop
- **Animações e transições** para uma experiência de utilizador fluída
- Links diretos para **WhatsApp** e **e-mail**

## 🚀 Como Executar Localmente

### Pré-requisitos

- Um browser moderno (Chrome, Firefox, Safari, Edge)
- [Git](https://git-scm.com/) instalado

### Passos

```bash
# 1. Clonar o repositório
git clone https://github.com/teu-usuario/alpainters.git

# 2. Entrar na pasta do projeto
cd alpainters

# 3. Abrir o ficheiro index.html no browser
# (ou usar a extensão Live Server no VS Code)
```

> 💡 **Dica:** Recomenda-se usar a extensão [Live Server](https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer) no VS Code para recarregamento automático durante o desenvolvimento.

## 🌍 Deploy

O projeto está hospedado na **Vercel** com deploy automático a partir da branch `main`.

```bash
# Instalar a Vercel CLI (opcional)
npm install -g vercel

# Fazer deploy
vercel --prod
```

Aceda ao site em produção: **[https://alpainters.vercel.app](https://alpainters.vercel.app)**

## 🏢 Sobre a Empresa

A **Alpainters** nasceu da paixão por transformar edifícios e garantir a sua durabilidade através de técnicas avançadas de alpinismo industrial. Os principais serviços incluem:

- 🏗️ **Isolamento Térmico** — Sistema de capoto e outras soluções energéticas
- 🏛️ **Reabilitação de Fachadas** — Preservação e renovação de edifícios
- 🎨 **Pintura Interior e Exterior** — Beleza e resistência com materiais ecológicos
- 🚁 **Vistoria Técnica com Drones** — Diagnósticos precisos de última geração
- 🧹 **Lavagem e Manutenção de Fachadas** — Limpeza e conservação especializada
- 🪟 **Limpeza de Vidros** — Acesso seguro em alturas de difícil acesso
